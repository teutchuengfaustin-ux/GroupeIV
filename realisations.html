Projet 1

Hotspot Wi-Fi avec Portail Captif

Système d'accès Internet contrôlé et sécurisé

Description du Projet

Conception et déploiement d'un système hotspot Wi-Fi professionnel avec portail captif personnalisé. Ce projet permet de gérer l'accès Internet d'utilisateurs via un système d'authentification flexible et une interface web personnalisée.

Objectifs Pédagogiques

  • Maîtriser la configuration avancée des routeurs MikroTik
  • Implémenter un système d'authentification réseau (RADIUS)
  • Personnaliser une interface web de portail captif
  • Gérer la qualité de service (QoS) et la bande passante
  • Assurer la sécurité du réseau sans fil

Technologies Utilisées

MikroTik RouterOS

Système d'exploitation pour routeurs MikroTik

Hotspot MikroTik

Module de gestion d'accès Wi-Fi avec authentification

User Manager / RADIUS

Système centralisé de gestion des utilisateurs

TCP/IP

Protocoles réseau pour la communication

HTML / CSS

Personnalisation du portail captif web

Résultats Obtenus

100%

Fonctionnalités implémentées

< 3s

Temps de connexion

50+

Utilisateurs simultanés testés

Projet 2

Système de Téléphonie sur IP (VoIP)

Architecture de communication voix sur réseau IP

Description du Projet

Mise en place d'un système complet de téléphonie IP basé sur FreePBX/Asterisk. Ce projet intègre la voix, la vidéo et les données sur une même infrastructure réseau avec qualité de service garantie.

Objectifs Pédagogiques

  • Comprendre les protocoles SIP et RTP
  • Administrer un serveur FreePBX/Asterisk
  • Configurer des téléphones IP et softphones
  • Implémenter la qualité de service (QoS)
  • Sécuriser les communications VoIP

Technologies Utilisées

FreePBX / Asterisk

PBX open source pour téléphonie IP

SIP, RTP

Protocoles de signalisation et transport voix

Linux (Ubuntu Server)

Système d'exploitation serveur

Téléphones IP

Matériel et logiciels clients VoIP

Sécurité VoIP

Chiffrement et authentification

Caractéristiques Implémentées

Projet 3

Authentification Biométrique par Reconnaissance Faciale

Système de sécurité innovant basé sur l'intelligence artificielle

Description du Projet

Développement d'un système d'authentification biométrique utilisant la reconnaissance faciale en temps réel. Cette application Python permet d'identifier et authentifier des personnes avec une haute précision grâce à des algorithmes de vision par ordinateur.

Objectifs Pédagogiques

  • Maîtriser le traitement d'images avec OpenCV
  • Implémenter des algorithmes de reconnaissance faciale
  • Développer une application Python complète
  • Gérer les bases de données d'empreintes faciales
  • Assurer la sécurité des données biométriques

Technologies Utilisées

Python 3.9+

Langage de programmation principal

OpenCV

Bibliothèque de vision par ordinateur

Dlib / face_recognition

Bibliothèques de reconnaissance faciale

NumPy

Calcul scientifique et manipulation de matrices

Caméra USB/Webcam

Capture vidéo en temps réel

Fonctionnalités Principales

Enregistrement

Capture et stockage des visages références

Reconnaissance

Identification en temps réel avec score de confiance

Anti-Fraude

Détection de photos ou masques

Journalisation

Logs détaillés des tentatives d'authentification
